Installing

comes as a Mac OS X installer package. Simply double-click .pkg to install. Please restart Terminal after installation to make sure that it inherits the new PATH environment variable.

does not include a text editor for editing source files. If you have Xcode already installed (e.g. because you also need to develop software for the Mac), simply use Xcode to edit your sources. If you don't have Xcode, recommend BareBone's TextWrangler instead. It's free. You may also use TextEdit (which ships with Mac OS X) for source code editing, but it lacks things like auto-indentation.

Upgrading

installs into /usr/local/-<version> and /usr/local/ is always a symbolic link to the version which has been installed last. Several versions of can therefore coexist and you can switch between them by changing the symbolic link. If you want only one version, remove the old installation before installing the new package, or delete it any time later from /usr/local/:
Shell Session:
bash$ cd /usr/local/ bash$ ls -ld * lrwxr-xr-x 1 root wheel 19 Dec 1 18:04 -> -20071201 drwxr-xr-x 15 root wheel 568 Nov 26 22:12 -20071126 drwxr-xr-x 18 root wheel 612 Dec 1 18:04 -20071201 bash$ sudo rm -rf -20071126 Password:

Removing

To remove completely, run the following command in a Terminal window:
Shell Session:
bash$ sudo /usr/local//uninstall Password: Are you sure you want to uninstall ? [y/N] yes Starting uninstall. is now removed.
